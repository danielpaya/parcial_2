Cómo montarlo en GitHub Pages
  1. Crea un repositorio en GitHub.
  2. Sube index.html, styles.css, app.js y questions.js.
  3. Entra a Settings → Pages.
  4. Selecciona Deploy from a branch, rama main y carpeta /root.
  5. Guarda y espera el enlace público.
